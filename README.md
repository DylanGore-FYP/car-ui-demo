# Car UI Demo

A proof of concept UI for the in-vehicle portion of the project - [https://github.com/DylanGore-FYP/Car](https://github.com/DylanGore-FYP/Car).

## Requirements

- Python 3
- [eel](https://github.com/ChrisKnott/Eel)

## Running

```bash
pip3 install -r requirements.txt
```

```bash
python3 demo_ui.py
```
